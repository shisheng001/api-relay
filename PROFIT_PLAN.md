## 二、盈利计算

**月流水估算（100用户，平均每天1000次调用）：**
- 平均每次调用消耗：输入5000 Token + 输出10000 Token
- qwen-turbo 单次成本：0.8×0.005 + 2×0.01 = **0.024元**
- qwen-turbo 单次收费：3×0.005 + 6×0.01 = **0.075元**
- **单次利润：0.051元**，利润率 **68%**
- 月利润估算：**100用户 × 1000次 × 0.051 = 5,100元/月**
